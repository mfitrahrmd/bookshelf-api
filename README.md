# bookshelf-api

Final project of "Belajar Membuat Aplikasi Back-End untuk Pemula" course provide by DTS FGA &amp; Dicoding Indonesia.

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run dev
```

### Compiles and start project

```
npm run start
```

### Lints and fixes files

```
npm run lint
```
